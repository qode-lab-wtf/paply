Einstellungen

API Keys
iKostenloser Key von console.groq.com. Wird für die Sprache-zu-Text Umwandlung mit Whisper Large V3 verwendet.
iOptional. Verbessert den transkribierten Text mit KI – korrigiert Grammatik und Zeichensetzung.
Optionen
iWenn aktiv, wird der Text nach der Transkription mit Claude Haiku aufbereitet. Benötigt einen Anthropic API Key.
iWenn aktiv, wird der Text automatisch ins vorherige Fenster eingefügt. Wenn deaktiviert, wird er nur in die Zwischenablage kopiert.
iSpielt einen kurzen Signalton ab, wenn die Aufnahme startet und endet.
iStartet die App automatisch, wenn du dich an deinem Computer anmeldest.
Transkription
iDie Sprache, in der du sprichst. Verbessert die Erkennungsgenauigkeit.
iGlobales Kürzel zum Starten und Stoppen der Aufnahme. Klicke ins Feld und drücke die gewünschte Kombination.
